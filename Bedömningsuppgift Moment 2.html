Bedömningsuppgift Moment 2

Frågor till Internet utveckling och webbhistoria:

  1. När startade utvecklingen av Internet? Vad startade utvecklingen?
    2. Utvecklingen av Internet började i slutet av 1960-talet med ARPANET, ett projekt finansierat av den amerikanska försvarsmyndigheten ARPA (Advanced Research Projects Agency). Målet var att skapa ett nätverk som kunde stå emot attacker och tillåta säker kommunikation mellan olika forskningsinstitutioner.
  2. Vad är fördelen med distribuerat nätverk i stället för centraliserat?
    3. I ett distribuerat nätverk är data och resurser spridda över flera noder, vilket ökar motståndskraften mot fel och attacker. Om en del av nätverket fallerar, kan andra delar fortsätta fungera, vilket minskar risken för total nätverkskollaps som kan inträffa i ett centraliserat system.
  3. Vad står WWW för?
    4. WWW står för World Wide Web, som är ett system för att länka och få tillgång till dokument på internet via hypertextlänkar.
  4. När blev Internet populärt för vanliga människor?
    5. Internet blev populärt för allmänheten i mitten av 1990-talet, särskilt med framväxten av användarvänliga webbläsare som Netscape Navigator och genom att fler hushåll fick tillgång till Internet via kommersiella internetleverantörer.
  5. Vilket är det grundläggande protokollet för att kommunicera över internet?
    6. TCP/IP (Transmission Control Protocol/Internet Protocol) är det grundläggande kommunikationsprotokollet som styr dataöverföring på internet.
  6. Vilket språk var föregångaren till HTML?
    7. SGML (Standard Generalized Markup Language) var föregångaren till HTML. HTML utvecklades som en enklare variant av SGML, anpassad för att skapa webbsidor.
  7. Vad var Gopher
    8. Gopher var ett tidigt informationssökningssystem som användes på internet före World Wide Web. Det byggde på en hierarkisk struktur av menyer och gjorde det möjligt att söka och hämta dokument.
  8. Vilka var de tre första webbläsarna?
    9. WorldWideWeb (senare kallad Nexus): Den första webbläsaren, utvecklad av Tim Berners-Lee 1990.
    Mosaic: En av de första webbläsarna med grafiskt gränssnitt, släppt 1993.
    Netscape Navigator: En mycket populär webbläsare lanserad 1994, som bidrog till att göra internet tillgängligt för allmänheten.
  9. Vad är en URL?
    10. URL (Uniform Resource Locator) är den unika adressen för en resurs på internet, såsom en webbsida, och specificerar var resursen finns och hur man kan komma åt den.
    T.ex. https://www.youtube.com/watch?v=0Qvm2-I9H40 vilket är en youtube video om hur man kan jobba i google classroom.
  10. När började vanliga personer att använda datorer?
    11. Persondatorer började bli vanliga i hushåll under 1980-talet, tack vare lanseringen av prisvärda och användarvänliga datorer som Apple II och IBM PC.
  11. Vad är syftet med CSS?
    12. CSS (Cascading Style Sheets) används för att styra utseendet och layouten på HTML-dokument. Det gör det möjligt att separera innehåll från design, vilket underlättar webbdesign och underhåll.
  12. Vad är XML? Och vad är skillnaden mellan XML och HTML?
    13. XML (Extensible Markup Language) är ett märkspråk som används för att strukturera, lagra och transportera data. Skillnaden mellan XML och HTML är att XML är designat för att vara flexibelt och användas för dataöverföring, medan HTML är specifikt utformat för att visa och formatera innehåll på webben.
  13. Vem uppfann Java? Vad hade Java för betydelse för Internet?
    14. Java utvecklades av James Gosling och hans team på Sun Microsystems, och lanserades 1995. Java gjorde det möjligt att skapa plattformsoberoende applikationer och applets, vilket var viktigt för att utveckla dynamiska och interaktiva webbsidor.
  14. Vad var en Applet (i Java-sammanhang)?
    15. En applet är ett litet program skrivet i Java som kan köras i en webbläsare. Applets användes ofta för att skapa interaktiva element på webbsidor, såsom spel, animationer och andra dynamiska funktioner.
  15. Vad är ett serverspråk? Vilka finns?
    16. Ett serverspråk är ett programmeringsspråk som körs på en server och hanterar logiken för dynamiska webbsidor, databasinteraktioner och liknande. Exempel på serverspråk inkluderar
    PHP
    Python
    C#
    Java
    SP.NET
  16. Vem skapade PHP?
    17. PHP skapades av Rasmus Lerdorf 1994. Det började som ett litet CGI-skript för att spåra besökare på hans personliga hemsida, men utvecklades senare till ett fullfjädrat serverspråk.
  17. Vad betyder ASP resp. JSP?
    18. ASP: Active Server Pages, ett ramverk utvecklat av Microsoft för att skapa dynamiska webbsidor.
    JSP: JavaServer Pages, en teknologi som används för att utveckla dynamiska webbsidor med hjälp av Java.
  18. Vad är Javascript? Vad är skillnaden på Javascript och serverspråken?
    19. JavaScript är ett skriptspråk som körs i webbläsaren och används för att skapa interaktiva element på webbsidor, som formulärvalidering, dynamiska menyer och effekter. Skillnaden mellan JavaScript och serverspråk är att JavaScript körs på klientens sida (i webbläsaren), medan serverspråk körs på servern och hanterar backend-logik och databasinteraktioner.

Frågor till lagar inom webbutveckling:

  1. Vad innebär copyleft?
    2. Copyleft är en licensmetod som tillåter fri användning, modifiering och distribution av ett verk, men kräver att alla modifierade versioner också distribueras med samma licens, vilket skyddar friheten för användare i framtiden.
  2. Vilka copyleft-licenser finns det?
    3. Exempel på copyleft-licenser inkluderar GNU General Public License (GPL), Affero General Public License (AGPL), och Lesser General Public License (LGPL).
  3. Vad har Copyleft för bild som symbol?
    4. Copyleft-symbolen är en spegelvänd version av copyright-symbolen ©, där det istället står "ↄ". Se bild
    Bilden är tagen ifrån Wikipedia
  4. Vad innebär Creative Commons?
    5. Creative Commons är ett system av licenser som ger upphovsmännen möjlighet att dela sina verk på villkor som de själva bestämmer. Licenserna erbjuder olika grader av frihet för användning, delning och modifiering av verken.
  5. Förklara Creative Commons-symbolerna?
    6. CC BY: Attribution – Verket får användas fritt, men upphovsmannen måste anges.
    CC BY-SA: Attribution-ShareAlike – Verket får användas fritt, men alla nya verk baserade på det måste ha samma licens.
    BY-ND: Attribution-NoDerivs – Verket får användas fritt, men får inte modifieras.
    CC BY-NC: Attribution-NonCommercial – Verket får användas fritt, men inte för kommersiella ändamål.
    CC BY-NC-SA: Attribution-NonCommercial-ShareAlike – Verket får användas fritt, men inte för kommersiella ändamål, och nya verk måste ha samma licens.
    CC BY-NC-ND: Attribution-NonCommercial-NoDerivs – Verket får användas fritt, men inte för kommersiella ändamål och får inte modifieras.
  6. Vilka är de stora skillnaderna mellan GDPR och föregångaren PUL?
    7. Den stora skillnaden mellan GDPR (General Data Protection Regulation) och PUL (Personuppgiftslagen) är att GDPR är en EU-förordning med starkare krav på samtycke, utökade rättigheter för individer att kontrollera sina data och kraftigare sanktioner vid överträdelser. GDPR gäller dessutom i hela EU, medan PUL var en nationell svensk lag.
  7. Om man hittar en bild på webben. Hur får man använda den bilden?
    8. Hur du får använda en bild beror på vilken licens bilden är under. Om bilden är upphovsrättsskyddad behöver du tillstånd från upphovsmannen för att använda den. Om den är licensierad under Creative Commons eller liknande kan du använda den enligt de villkor som anges, såsom att ange upphovsmannen eller att inte använda den kommersiellt. Om bilden är i public domain kan den användas fritt.
